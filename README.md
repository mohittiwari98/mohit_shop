## 📸 Preview

> A fast, modern and responsive shopping cart app built with the latest React ecosystem.

---

## ✨ Features

- 🛒 **Shopping Cart** — Add, remove and manage products seamlessly
- 📱 **Fully Responsive** — Works beautifully on mobile, tablet and desktop
- ⚡ **Lightning Fast** — Powered by Vite 8 for instant HMR and blazing builds
- 🎨 **Beautiful UI** — Styled with Tailwind CSS v4
- 🔔 **Toast Notifications** — Real-time feedback with React Hot Toast
- 🔗 **Client-side Routing** — Smooth navigation with React Router v7
- 🚀 **Auto Deploy** — CI/CD via GitHub Actions to GitHub Pages

## 🚀 Getting Started

### Prerequisites

![Node](https://img.shields.io/badge/Node.js-20+-339933?style=flat-square&logo=node.js&logoColor=white)
![npm](https://img.shields.io/badge/npm-latest-CB3837?style=flat-square&logo=npm&logoColor=white)

### Installation

```bash
# Clone the repository
git clone https://github.com/mohittiwari98/mohit_shop.git

# Navigate to client folder
cd mohit_shop/client

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
